# MNIST Digit Classification using CNN

## Project Overview
This project uses a Convolutional Neural Network (CNN) to classify handwritten digits (0–9) using the MNIST dataset.

## Dataset
https://www.kaggle.com/datasets/oddrationale/mnist-in-mnist

## Model Architecture
- Conv2D layers
- MaxPooling layers
- Dense layers
- Softmax output

## Results
- Test Accuracy: ~99%

## Tools Used
- Python
- TensorFlow / Keras
- NumPy
- Pandas

## How to Run
1. Open notebook
2. Run all cells
3. Ensure dataset is loaded properly
